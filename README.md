# AI Company Intelligence Tool

This project is an AI-powered automation tool that analyzes companies and their market positioning.

## Features

- Finds a company's official website automatically
- Scrapes website content
- Cleans and processes text data
- Uses AI to generate a structured company analysis
- Identifies business strengths and weaknesses
- Detects potential automation opportunities
- Finds and lists competitors
- Saves results to CSV

## Tech Stack

Python  
OpenAI API  
BeautifulSoup  
SerpAPI
